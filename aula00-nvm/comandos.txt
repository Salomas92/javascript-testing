nvm install 20 -> instala a versão 20
nvm install --lts -> instala a versão long-term support (versão recomendada)
nvm list -> lista todas as versões disponiveis
nvm use 18 -> usa a versão 18 que ja está instalada na minha maquina
nvm alias default 18 -> define a versão padrão do Node.js para ser a 18

criar .nvmrc - com a versão desejada e nvm use para ativar aquela versão
node -v .nvmrc -> salva a versão apropriada para o aquele projeto
